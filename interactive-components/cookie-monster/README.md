# Cookie Monster - Interactive Eye Tracking Demo

A web implementation featuring eye-tracking mechanics with cross-eye effect.

## Live Demo
- GitHub Pages: https://sulakova-s.github.io/frontend-practice/

## Key Features
- Eye tracking - Pupils follow cursor movement
- Cross-eye effect - Eyes converge when cursor is close to face
- Optimized performance - Smooth animations
- Responsive design - Adapted for mobile (320px+) and desktop
- Accurate design - Matches original Figma layout

## Tech Stack
Frontend: HTML5, CSS3, Vanilla JavaScript (ES6+)

## Project Structure
src/
├── index.html          # Semantic HTML structure
├── style.css           # Styling with CSS custom properties
├── script.js           # Eye tracking algorithm
└── resources/
    └── cookie.png      # Icon

## Quick Start
1. Open `index.html` in a modern browser
2. Move cursor around the page
3. Eyes will follow your movement
4. Approach the center to see cross-eye effect
